


# HOW TO INSTALL
Disclamer: You will obviously need [Python3](https://www.python.org/downloads/) and [pip](https://pip.pypa.io/en/stable/installation/) installed.
OR YOU CAN JUST DOWNLOAD THE .EXE IN RELEASES

1. Clone the repo  
```git clone https://github.com/costeadev/camera-lol.git```

2. Move into the folder cloned  
```cd camera-lol```

3. Create a new virtual environment  
```python -m venv .venv```

4. Activate the virtual environment  
Windows: ```.venv\Scripts\activate```  
Linux: ```source .venv/bin/activate``` or just ```.venv\bin\activate```

5. Install the requirements (make sure you have your venv activated beforehand)
```pip install -r requirements.txt```

6. Run the program (make sure you have a camera plugged in)
```python main.py```
